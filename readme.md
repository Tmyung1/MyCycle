# 🌸 My Cycle

**A beautiful, private & calming menstrual cycle tracker** built with PyQt6.

Track your cycle, moods, intimacy, diary entries, and daily moments with a soft pastel aesthetic and thoughtful details like moon phases and local weather.

</div>

## ✨ Features

- **Monthly Calendar** with rich visual indicators:
  - Period days (🩸)
  - Fertile window (🥚)
  - Intimacy marker (❤️) — drag heart stickers!
  - Mood-colored days
  - Diary entries (✏️)
  - Moon phase icons

- **Daily Journaling**
  - Mood selector (10 levels)
  - Rich text diary
  - Drag & drop daily snapshot images

- **Cycle Insights**
  - Next period & fertile window predictions
  - Cycle history tracking

- **Atmosphere**
  - Real-time local weather
  - Accurate moon phase calculations
  - Smooth animations and soft pastel design

- **Desktop Experience**
  - System tray support
  - Single instance enforcement
  - Works offline after first launch
